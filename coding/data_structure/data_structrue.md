### 算法执行效率及资源消耗
#### 大O复杂度表示方法
> 读数据 - 运算 - 写数据；
> 所有代码执行时间T(n)与每行代码的执行次数n成正比；

#### 时间复杂度分析
> 只关注**循环执行**次数最多的一段代码；
> 加法准则：总的时间复杂度等于量级最大的代码的时间复杂度;
``` O(n*n) = O(n*n) + O(n) + O(1)
```
> 乘法准则：嵌套代码的复杂度等于嵌套内外代码复杂度的乘积；
